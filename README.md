# CRM-system
Схема архитектуры CRM-платформы фитнес-клуба, диаграмма потоков данных (DFD) для процессов управления клиентами и тренировками, схема базы данных (ER-диаграмма) фитнес-CRM, макеты пользовательского интерфейса (wireframe) ключевых экранов, визуализация отчетов по загрузке тренеров (диаграммы и графики).
